DISPLAY

No cambia el tipo de elemento,cambia cómo se visualizan los elementos en mi sitio web

Los tipos de lementos html:

  • Bloque: block, elemento que empieza en una nueva línea,ocupa todo el ancho disponible de la pantalla,a menos que le indiquemos tamaño con width
    1. Encabezados h1-h6
    2. párrafo
    3. formulario
    4. lstas ul,ol.dl, y sus li
    5. div
    6. table,td,tr,th
  • En línea: inline,ocupan solamente el tamaño de su content,no les podemos dar ancho ni alto, y van uno a continuación del otro
    1. a enlaces
    2. imágenes
    3. span

VALORES DEL DISPLAY

  • none: no se muestra.No deja el hueco del elemento que se oculta
  • block: se muestra como si fuera un elemento en bloque
  • inline: se muestre como si fuera un elemento inline
  • inline-block: se muestra como si fuera medio bloque medio inline
